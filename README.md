## Modelo de Excel recomendado

1ª linha: número das questões: Q1, Q2, Q3...

2ª linha: descritores de cada questão: D1, D2, D3...

3ª linha: gabarito: A, B, C, D, E...

Demais linhas: respostas dos alunos.

Exemplo:

| Aluno | Q1 | Q2 | Q3 |
|---|---|---|---|
| Descritores | D16 | D4 | D4 |
| Gabarito | C | C | C |
| Ana Silva | C | D | B |
